Pregunta: ¿Qué es Python?
Respuesta: Python es un lenguaje de programación de alto nivel, interpretado y orientado a objetos. Es ampliamente utilizado en desarrollo web, análisis de datos, inteligencia artificial y automatización.

Pregunta: ¿Qué es una API?
Respuesta: Una API (Application Programming Interface) es un conjunto de definiciones y protocolos que permite que diferentes aplicaciones se comuniquen entre sí.

Pregunta: ¿Qué significa REST?
Respuesta: REST (Representational State Transfer) es un estilo arquitectónico para el diseño de servicios web que utiliza operaciones HTTP estándar como GET, POST, PUT y DELETE.

Pregunta: ¿Qué es una base de datos relacional?
Respuesta: Una base de datos relacional organiza los datos en tablas con filas y columnas, y permite establecer relaciones entre ellas mediante claves primarias y foráneas.

Pregunta: ¿Qué es Docker?
Respuesta: Docker es una plataforma que permite crear, implementar y ejecutar aplicaciones en contenedores, lo que garantiza portabilidad y consistencia en distintos entornos.
